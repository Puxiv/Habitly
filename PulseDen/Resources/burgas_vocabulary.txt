Бургаски Говор (Източен / Крайморски) – Речник и Граматика (Approved Vocabulary)
Диалект „Батка" – Бургас / Черноморието

== ГРАМАТИКА — Задължителни правила ==

Местоимения:
аз = аз (стандартно)
ти = ти
той = той
тя = тя
ние = ние
вие = вие

Характерни фонетични особености:
1. „е" от ят (ѣ): бел (бял), хлеб (хляб), млеко (мляко), сено (сяно) — подобно на западните говори
2. Типично за крайбрежието: по-меко, по-протяжно произношение
3. Изпускане на „х" в началото: убаво (хубаво), айде (хайде), ора (хора)
4. „ш" вместо „щ" в някои думи: нош (нощ), пуш (пущам)
5. „кво" вместо „какво"
6. Честа употреба на „батка", „ей", „де", „море"
7. Протяжно „е" и „о" в края на изречението

Типични глаголни форми:
ще + глагол (стандартно)
видел = видял
отишел = отишъл
рекел = рекъл (казал)

== Характерни думи и изрази ==

батка = бате, братко (типично бургаско обръщение)
море = възклицание (от морския град, „Море, чуй!")
ей = частица за обръщение
де = частица (Де да е, де!)
па = пък, а (Па ти кво?)
кво = какво
убаво = хубаво (без „х")
айде = хайде
арно = добре
бая = много, доста
мальо = малко
тъй = така
кат = като (Кат ти казвам!)
хич = изобщо, никак
ей сега = точно сега, веднага
барабар = заедно (от турски)
саат = час (от турски)
гьонсурат = безочлив, нахален
алабрак = безредно, без ред
хамсия = дребна риба / прен. дребен човек
марина = пристанище
лиман = залив / пристанище
балък = риба (от турски)
калайджия = калайджия
чаршия = пазар
кеф = удоволствие, настроение
рахат = спокойствие, комфорт
хатър = уважение, респект
мерак = страст, желание, копнеж
касавет = грижа, тревога
дерт = мъка, болка
занаят = занаят, умение

== Обръщения ==

батка = към мъж (основно обръщение)
море = емоционално обръщение (Море, батка!)
бе = за емфаза
сестро = към жена (приятелски)
комшу = съсед
аго = към по-възрастен

== Морски / крайбрежни думи ==

хамсия = дребна риба (и прен. за дребен човек)
цаца = друг вид дребна риба
попара = ястие от хляб
чушкопек = любимо занимание на бургазлии
рибата = „на рибата" = отиваме на риба
вълна = морска вълна
бриз = морски бриз
пясък = на плажа

== Поговорки и типични фрази ==

Море, батка, кво убаво е! = Колко е хубаво!
Ей, батка, чуй! = Чуй ме!
Арно де, арно = Добре де, добре
Бая жега, айде на морето! = Много е горещо, да идем на морето!
Хич не ме е грижа = Изобщо не ме е грижа
Кат ти казвам, тъй е! = Както ти казвам, така е!
Ей сега ще дойда = Веднага ще дойда
Рахат да ти е = Да ти е спокойно
Мерак ми е за морето = Копнея за морето
Море, хамсията пак удари = Хамсията пак е дошла (рибата)
Батка, на рибата ли? = Отиваш ли на риба?
Барабар ще ходим = Заедно ще ходим
Хатъра ти го имам = Уважавам те / ще ти направя услуга
Гьонсурат е тоя = Безочлив е този
