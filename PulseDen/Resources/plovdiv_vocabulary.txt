Пловдивски Говор (Тракийски) – Речник и Граматика (Approved Vocabulary)
Диалект „Майна" – Пловдив / Тракия

== ГРАМАТИКА — Задължителни правила ==

Местоимения:
аз = аз (стандартно, но често се изпуска)
ти = ти
той = той
тя = тя
ние = ние (но „ний" в по-стар говор)
вие = вие

Характерни фонетични особености:
1. Широко „а" от ят (ѣ) под ударение: хл'аб (хляб), мл'ако (мляко), б'ал (бял), с'ано (сяно)
2. „е" от ят без ударение: хлебар (хлебар), белина (белина)
3. Меки съгласни пред широко ят: м'асто (място), д'адо (дядо)
4. Често изпускане на „х": убаво (хубаво), ора (хора), айде (хайде)
5. „кво" вместо „какво"
6. „тъй" вместо „така"
7. Честа употреба на частиците „бе", „ми", „ей", „майна"

Типични глаголни форми:
ще + глагол (стандартно, както в книжовния)
видел = видял
дошел = дошъл
казвам = казвам (стандартно)

== Характерни думи и изрази ==

майна = възклицание за учудване, одобрение, недоволство (универсално пловдивско)
бе = частица за обръщение и емфаза (Е бе, кво стана?)
убаво = хубаво (без „х")
кво = какво
тъй = така
ей = частица за обръщение (Ей, чуй!)
ми = частица (Ами ми, тъй е.)
пусто = проклето / интензификатор (Пусто, жега е!)
стига бе = стига, недей
сабале = сутрин (рано сутринта)
вечерта = вечерта (тази вечер)
нагоре-надолу = нагоре-надолу (по Главната улица)
арно = добре
мальо = малко
бая = много, доста
тепе = хълм (от 7-те пловдивски тепета)
капия = врата, порта (от старинния Пловдив)
маане = махала
ракийка = ракия (умалително, типично тракийско)
акъл = ум, разум (турски произход)
алъш-вериш = търговия, размяна
кефа = удоволствие, настроение
чорба = и буквално, и прен. „каша, бъркотия"
салтанат = парадност, великолепие
бакалин = бакалия / магазинче
саат = час / часовник
халал = безплатно / заслужено (Халал да ти е!)
хич = изобщо, никак
сукалче = улица (от „сокак")
чаршия = пазар, търговски център

== Обръщения ==

бе = към всеки (Е бе, ти кво?)
майна = самостоятелно или в обръщение (Майна, кво стана!)
чорбаджи = към по-заможен (шеговито)
комшу = съсед (турски произход)
аго = към по-възрастен мъж

== Поговорки и типични фрази ==

Майна, кво убаво е! = Колко е хубаво!
Арно бе, арно = Добре де, добре
Пусто, жега е! = Проклетата жега!
Ей, стига бе! = Хайде, стига!
Бая работа има = Много работа има
Тъй да е, тъй ще е = Така е, така ще бъде
Нагоре-надолу по Главната = Разхождане по Главната улица
Хич не го бива = Изобщо не е добро
Халал да ти е! = Нека ти е заслужено / наздраве!
Кефа ми стана = Стана ми приятно
Сабале рано, вечерта тъмно = Сутрин рано, вечер тъмно
На тепето е убаво = На хълма е хубаво
Майна, ти бая акъл имаш! = Ти си много умен!
