Врачански диалект – Одобрен речник (Approved Vocabulary)

== Обиди и определения за хора (Insults / Descriptions of People) ==

омраз = омразен (долен тип)
поган = езичник / чужденец
плякор = паплач, сган, тълпа
келеметор = паплач, бедни хора, сган
копилитор = копеле
дзвер = звяр (може положително или отрицателно)
изклесък = изтърсак (универсална обида)
прошляк = нещастник, бедняк
пелентозавър = митично животно; обида за грозник
плашипутарник = много грозен човек
лайновоз = дебелак (обидно)
ботур = пън / темерут, груб недодялан човек (Кво глеаш като ботур бе!)
ръняв = грозен (силно експресивно)
киък / кияк = смотаняк (Дееба тоя киък!)
абич / абен = некадърен (Могу си па абен бе!)
пукница = пияница (Голема пукница си!)
цинцифуция = префърцунен, неразбран, претенциозен човек
гръчав = слаб (Жено тизе гръчава!)
лефтерен = слаб / паянтов, чуплив
неалит = калпав / лош (Могу си неалит бе.)
абдал = глупак (турцизъм)
джинс = род (Джинс мръсен!)
клен = непохватен / глупав
скраклюзен = омърлушен, жалък
повлеч = раздърпан / смотан
оръпляк = смотан / грозен
боцуняк = намусен / темерут (Кво глеаш кат некой боцуняк.)
скучубра = грозна жена
ламер = груб / непохватен
умреляк = не е „по живота" (Кьъф е тоа умреляк бе?) силно презрително
некролог = старец
турта = пита / дебела жена
клюнор = голям нос (Пепа Клюнора)
виден охлюв = известен персонаж (ироничен израз)

== Местоимения и граматика (Pronouns & Grammar) ==

язе / я = аз (Язе, тизе и момчеата с колелеата.)
тизе = ти
тава = това
он ин ги е баща = той им е баща (характерна конструкция)

== Глаголи (Verbs) ==

яла = ела (яла сам – ела тук)
онодвам = правя (общ смисъл)
преонодвам = преправям
зея / зеиш = викам / крещя (Кво сса раззеяла ма!)
сакам = искам (Кнежъ, Враца, Бела Слатина)
очем = искам (около Лом и на запад)
спрепам се = препъвам се (Кво са спрепаш бе!)
совнах са = паднах / тръгнах (многозначно)
изпуизпръзулих са = подхлъзнах се
чмъ / чмиш = спя / спиш
пръам = кихам
рескам яко = пия много
набабиникям = насищам се (нахранвам се или се напивам)
джумоля = мърморя (Кво ми джумолиш!)
очеплезил = облещил (многозначно)
чеплезя се = лигавя се, държа се неприлично
рипа ми гъзъ = не ме интересува
тура го на майкя му = не ме интересува
връва с клоняка = вървя грубо, блъскам се
връва на главътъ си = вървя без да се съобразявам
да се съсипя = да се разваля (Млекото саа съсипало.)
да се сбабичасам = сбръчква ми се кожата
отпарай = махай се, изчезвай
дръпай = махай се (заплашително)
рампраш = махай се
зашлекнах са = задавих се
забакнах са = задавих се
сплесках кофата = отслабнах
маиняй са = махай се (Аде майняй са!)
да пъкна = да се покажа (Пъкни ма ооу, яла а ти кажа нешту.)
варнах го = изпреварих го
аа са варшкаме = да се надпреварваме
зобам = ям
плюскам = ям
кусам = ям с лъжица
громам = вдигам шум / тропам
да са натрепа = да се нахраня хубаво
да са съпикясам = да се осъзная / въздържа
да се събедя = да се осъзная (Де събеди са бе!)
заеби = забрави
дигай гълъбете = изчезвай

== Съществителни (Nouns) ==

пароляк = пари
ботур = пън (преносно: груб човек)
кукуруз = царевица
лебеница = диня
кръшняк = кръст/анатомия (Заболе ма кръшняка от работа.)
пръделник = задник
ала = хала / змей (Пиеш като борованска ала.)
клепаре / клаповци = стари чехли
кутлеварка = метален съд за мляко или вода
казма = права лопата (Подай ми казмътъ.)
наджак = брадва
манара / манарче = брадва / брадвичка
балтия = малка брадва
цекленка = стъклено шише за ракия
перашки = боядисани великденски яйца
поворка = наниз (най-често сушени люти чушки)
бел лук = чесън
шъпляк / бебек / ломняк = камък
барабиняк / барабинци = мравка / мравки
ендък = дупка
порезаница = дебела филия хляб
черпиняк = парцал
стока = домашни животни (Да дам на стоката.)
дживгар = врабче

== Наречия и частици (Adverbs & Particles) ==

лазда / лаас = може би / сигурно
оти = защо
отвъде / удвъде = оттатък (в другата стая)
удоздоле = отдолу
удозгоре = отгоре
надвър = навън
у дънъка = оттатък
у точка = преди малко
чако / дако = нищо / какво от това (не ми пука)
боли ма уя = не ме интересува
дреме ми на сайдера = не ме интересува
за кашмер = много зле (турцизъм; и приятелско)
е те тукеенкъна те = ето тук

== Възклицания и изрази (Exclamations & Phrases) ==

маалиии = възхищение при инцидент или изненада
дии / дееба = възхищение / изненада
желееее = силна емоция
кеф на буци = голяма радост
кеф на бучки, зор на каручки = безсмислица
е пъ къ / ми къ = е пък как / ами как (силна увереност)
еш ги па таа = виж я пък тази
еле го е / е го е = ето го и този (очакван)
И те така те… = израз за край на разговора
живота е сурав и курав = северо-западна сентенция
най обичам леба и да еба = северо-западна житейска философия
наглъфал съм се като бибе с дуди = преял съм
налапал си се с лафове като бибе с цифки = много знаеш
пълен си с акъл като магаре с пръдни = много си умен (иронично)
мааакя ма / мааре = наистина ли? учудване
ша еба да… = ще взема да… (модален глагол)
нема да та забава = иронично – няма намерение да изпълни нищо

== Обръщения (Address Forms) ==

баце = обръщение към всеки (положително или пренебрежително)
жено мънинка / жено малка = към всичко живо (гальовно или заплашително)
еее, мъдестия = подигравателно обръщение
дуу жено де = по-полека, не се изсилвай

== Пожелания и поздрави (Greetings & Wishes) ==

леко а съ не осереш = вместо „Наздраве!" когато някой кихне
връй пикай, па легай = вместо „Лека нощ"

== Сравнения и метафори (Comparisons) ==

кво се моташ като глуа кучка из зелен кукурузляк = скита се безцелно
омърлушил си се като ваксинирана кокошка = изглеждаш мизерно
одиш като кьорав конь из гробища = вървиш неловко
пееш като магаре у падина = пееш гръмогласно и фалшиво
кво висиш като пуяк на пръщина = стоиш и гледаш без да помагаш
кво са влачиш като свинско чирво = вървиш бавно
улете като манарче у гнила тиква = влезе лесно без пречки
наглъфал съм се като бибе с дуди = преял съм
пълен си с акъл като магаре с пръдни = много си умен (иронично)
кух на градинска лейка = глупав / тъп
прост на кантонерски петел = много прост
шашав по целата глава = луд / избухлив
пиеш като борованска ала = пиеш много
глей ка са носи кат чиресло на ветар = гъзарее се

== Фонетични бележки (Phonetic Notes) ==

Подвижно ъ: влък → влъка, връба → вр'ъба (в групите ър, ъл)
Палатализация на „н": сиреньето, бидоня, телефоня, коньете
Ударение: глаголът еба винаги с ударение на „е"
