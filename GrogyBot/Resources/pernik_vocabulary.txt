Пернишко-Шопски Говор – Речник и Граматика (Approved Vocabulary)

== ГРАМАТИКА — Задължителни правила ==

Местоимения:
яз = аз (основна форма, ВИНАГИ)
он = той
она = тя
оно = то
ние / ни = ние
вие / ви = вие

Бъдеще време — частица „ЧЕ" (НЕ „ще"):
яз че идем = аз ще дойда
он че каже = той ще каже
че те питам = ще те питам
че видиш = ще видиш
че му дадем = ще му дадем
Пример: „Яз че говорим, ти само че мълчиш и че кимаш."

Фонетика — ят (ѣ) → само „Е" (никога „Я"):
бел = бял
бело = бяло
лево = ляво
снег = сняг
цвет = цвят
место = място
месец = месец

Окончания и форми:
ч вместо шт: ноч (нощ), свеч (свещ), вреч (врещ)
дж вместо жд: меджу (между), роджен (роден)
у от старото носово: зуп (зъб), мука (мъка), рука (ръка), нугу (крак)
Членна форма -о за мъжки род: човеко, путето, братко
Множествено число с -ете: будалете (будалите), вратарете

== Глаголи (Verbs) ==

клендза = куцука (Он клендза по путето.)
яя / яздя = яздих (Я ю яя. = Аз я яздих.)
емчи се = напъва се (Емчи се ко вол у яръм.)
вали се = хвали се (Кой се млого вали...)
опраймо = оправим (Са да си опраймо есапите.)
едомо = ядохме (Синочка едомо качемак.)
блюи = бълва (за вода)
дурляе = тече монотонно
църцори = църкори
са = сега (Са да си...)
синочка = снощи
дека = накъде / откъде
декате = скарвам се, държа се назидателно
бреца = досажда, истерично говорене

== Съществителни (Nouns) ==

путето = пътчето (членувано с -о)
яръм = ярем
мас = лой / масло (Живее ко бубрег у мас.)
бубрег = бъбрек
качемак = качамак
жмърци = пръжки
есап = сметка (от турски)
дуня = свят / хора (Дунята че се сврши...)
будале = будали (мн.ч. будалете)
слода = сговор, доверие, приятелство и взаимност
лефтер = спретнат (само в трънския говор)
бриди = духа студен вятър
зуп = зъб (старо носово у)
рука = ръка
нугу / нога = крак
ноч = нощ (ч вместо шт)
свеч = свещ
снег = сняг

== Наречия и частици ==

са = сега (много често)
синочка = снощи
млого = много
ко / като = като (Живее ко бубрег у мас.)
у = в (предлог)

== Изрази и поговорки ==

Он клендза по путето = Той куцука по пътчето
Я ю яя = Аз я яздих (кобилата)
Емчи се ко вол у яръм = Напъва се като вол в ярем
Живее ко бубрег у мас = Живее охолно (като бъбрек в лой)
Кой се млого вали, он не пали = Който много се хвали, не върши работа
Са да си опраймо есапите = Сега да си оправим сметките
Синочка едомо качемак с жмърци = Снощи ядохме качамак с пръжки
Дунята че се сврши, будалете че остану = Светът ще свърши, будалите ще останат
Водата тече: блюи, дурляе, църцори, капе = Водата тече: бълва, тече, църкори, капе
Яз че говорим, ти само че мълчиш и че кимаш = Аз ще говоря, ти само ще мълчиш и ще кимаш
Огняне, че ти го обясним на пернишки = Огняне, ще ти го обясним ясно
